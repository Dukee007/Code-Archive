    • Command Discription Usage
    lock Locks a channel from normal users .lock
    unlock Unlocks a channel from normal users .unlock
    report Allows you to report users for a reason (only if setup by server admins use .reportsetup in the channel for reports) .report {who} {reason}
    serverinfo Show infomation about the server .serverinfo
    resetmessages Delete's all messages in every channel .resetmessages
    clearuser Delete's all messages sent by the given user in every channel .clearuser {user}
    raw Show the raw data of a message .raw {data}
    clear Allows you to clear a amount of messages in a channel .clear {ammount/user}
    mute Allows you to mute a user .mute {user}
    unmute Allows you to unmute a user .unmute {user}
    kick Allows you to kick a user .kick {user}
    ban Allows you to ban a user .ban {user}
    reportsetup Allows staff to setup reporting for your server! type in the report log channel. .reportsetup
    reportreply Allows server staff to respond to user reports .reportreply {target/sender} {message} {report_id}
    report Allows you to report users for a reason (only if setup by server admins use .reportsetup in the channel for reports) .report {user} {reason}
    play Play a song from youtube or resume a paused song .play {url/search}
    pause Pause a playing song .pause
    join Makes the bot join a channel or join the channel your in .join
    leave Make the bot leave the channel its currently in .leave
    stop Leave's the channel and clears the queue .stop
    np Allows you to see the song the bot is currently playing .np
    queue Allows you to see the queue .queue
    skip Allows you to skip the current song .skip
    translate Translate text to a diffrent langage .translate {country_code} {text_to_translate}
    talk The bot will talk for you in a diffrent langage .talk {country_code} {text_to_say}
    remove Removes any song form the queue .remove {song_number}
